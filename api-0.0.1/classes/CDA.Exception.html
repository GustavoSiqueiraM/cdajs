CDA.Exception Class

Defined in: lib\cda.js:304
Module: CDA.js

This class is used to indicate an runtime errors occurring in any of the methods of the cda.js classes.

You do not need to instantiate objects of this class yourself. Rather, instances of this class are created and thrown at runtime whenever an error occurs.

To handle CDA.js errors, you can use a try...catch block like this: 

   try {
       ...general cda.js work...
   } catch (exception) {
       if (exception instanceof CDA.Exception) {
           ...use exception.code, exception.message and exception.data to handle the exception.
       } else {
           ...handle other errors...
       }
   }
